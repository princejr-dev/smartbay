# SmartBay

SmartBay est une application web de gestion locative conçue pour aider les propriétaires à suivre leurs locataires, organiser les loyers et générer des reçus de paiement.

---

## 🚀 Fonctionnalités principales

- Gestion des locataires (ajout, modification, suppression)
- Suivi des informations de location
- Génération de reçus de paiement
- Organisation et filtrage des locataires
- Recherche rapide par nom
- Interface moderne et responsive (mobile / desktop)

---
